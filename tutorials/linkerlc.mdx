---
title: "Linking your ERLC Private Server"
description: "Learn how to link your ERLC private server properly, allowing you to manage, monitor, and automate it remotely."
---

<Warning>
  You must be the owner of the ERLC private server in order to access the private server settings necessary for this configuration.
</Warning>

<Steps>
  <Step title="Link your ERLC Private Server">
    1. Join your ERLC private server.
    2. Press the server stack icon on the top right corner of the screen.
    3. Select **Edit Server Settings**.
    4. Search for **API Key**.
    5. Press **Edit** and copy the API key.
    6. Run the **`/setup`** command and navigate to the **ERLC Integration** page.
    7. Select **Edit API Key** and paste the key you copied.
    8. Press **Submit**, and your server should be linked.
  </Step>
  <Step title="Completed">
    You have now successfully linked your ERLC private server to Ducky.
  </Step>
</Steps>

<Warning>
  If your API key does not seem to work, reset it within ERLC and re-enter it. If you have any problems, feel free to open a support thread in [**Ducky's Pond**](https://discord.gg/vUHzCTNetw)**.**
</Warning>
