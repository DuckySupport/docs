---
title: " Setting up Ducky"
description: "Learn how to set up  Ducky in a couple of steps."
---

Once you have invited Ducky to your server, use the **/setup** command to start configuring your server. From there you can check out and configure some of the many pages ducky has to offer. 

<Warning>
  All members with the **Manage Server** permission can access the

  Ducky **Setup** page.
</Warning>
