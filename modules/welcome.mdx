---
title: "Welcome/AutoRoles"
description: "Ducky's Welcome/AutoRoles module allows for you to send a message to welcome your users in the specified channel, as well as give them up to 5 roles automatically."
---

<Steps>
  <Step title="Autoroles" iconType="regular" titleSize="h2">
    - You can select up to 5 roles that gets automatically added to a user once they join the server.
  </Step>
  <Step title="Welcome Channel" titleSize="h2">
    - You can select a channel where the Welcome Message gets sent to.
  </Step>
  <Step title="Welcome Message" titleSize="h2">
    - You can configure the Welcome Message here, you can choose if you want an embed or not. There are also a list of variables which can enhance the message.
  </Step>
</Steps>
