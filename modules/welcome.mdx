This page provides the essential configuration options for **Ducky's Welcome/AutoRoles** module, which allows you to greet new members and automatically assign roles. The module creates a centralized system for welcoming users with a custom message in a specified channel and assigning up to five roles automatically. To enable and configure this module, use the `/setup` command and navigate to the **Welcome/AutoRoles** page.

<Steps>
  <Step title="Autoroles" iconType="regular" titleSize="h2">
    **Auto-Assign Roles** Allows you to select up to 5 roles that are automatically assigned to a user when they join the server.
  </Step>
  <Step title="Welcome Channel" titleSize="h2">
    **Welcome Message** Lets you choose a channel where the welcome message will be sent.
  </Step>
  <Step title="Welcome Message" titleSize="h2">
    **Welcome Message Configuration** Allows you to customize the welcome message, choose whether to use an embed, and enhance the message with a list of available variables.
  </Step>
</Steps>
