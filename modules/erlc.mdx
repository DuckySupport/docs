This page provides the essential configuration options for Ducky's ERLC Integration module, which enables seamless management and monitoring of your ERLC server without needing to join the game. To enable and configure this module, use the `/setup` command and navigate to the ERLC Integration page.

<Steps>
  <Step title="In-Game Role" titleSize="h2">
    The **In-Game** **Role** is a configurable role added to the discord member upon joining the linked ERLC Server.

    The role is removed from the member when the player leaves the server.

    <Warning>
      This requires that the player is linked with Ducky.
    </Warning>
