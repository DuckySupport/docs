title: "1. Welcome To Ducky"
description: "Learn what this module does and what you can do with it!"
---

This page provides the essential configuration options to help you get started quickly, including setting your command prefix, managing disabled commands, adjusting permissions (see [Permissions](https://docs.duckybot.xyz/configuration/permissions) for details) , configuring a log channel, and resetting your configuration.

<Steps>
  <Step title="Prefix" titleSize="h2">
    - The **Prefix** determines how commands are triggered in your server.
    - You can change it on this page to fit your server's style.

    <Info>
      This option only applies to text-based commands, slash commands still work the same\!
    </Info>
    - 
  </Step>
  <Step title="Disabled Commands" titleSize="h2">
    - This option allows you to disable specific Ducky commands you don't want available in your server.
    - Disabled commands will not function when triggered/used.
  </Step>
  <Step title="Permissions" titleSize="h2">
    - Configure which roles can access certain commands.
    - For more detailed information on permissions, see [Permissions](https://docs.duckybot.xyz/configuration/permissions).

    <Tip>
      Restrict sensitive commands to trusted staff roles only.
    </Tip>
  </Step>
  <Step title="Configuration Log Channel" titleSize="h2">
    - This is where all configuration changes made to Ducky will be logged.

    <Tip>
      Use a private staff channel so only authorized members can view configuration updates.
    </Tip>
  </Step>
  <Step title="Wipe Configuration" titleSize="h2">
    - This option allows you to **reset all Ducky settings back to default**.
    - Use this only if you want to completely reconfigure from scratch.

    <Warning>
      Wiping the configuration is **permanent** and cannot be undone\!
    </Warning>
  </Step>
</Steps>
