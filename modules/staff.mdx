This page provides the essential configuration options for Ducky's **Staff Management** module, which allows you to manage your staff team effectively. The module creates a centralized system for issuing infractions, handling promotions, and maintaining staff records, ensuring your team is organized and accountable. To enable and configure this module, use the **`/setup`** command and navigate to the **Staff Management** page.

<Steps>
  <Step title="Infractions Channel" titleSize="h2">
    The **Infractions Channel** is a designated channel where all issued infractions are sent to. A user receives an infraction when an **ERLC Manager** runs the `/infraction issue` command.
  </Step>
  <Step title="Promotions Channel" stepNumber={2} titleSize="h2">
    The **Promotions Channel** is a designated channel where all promotion messages are sent. A user can be promoted when an **ERLC Manager** runs the `/promote` command. The promotion embed can also be customized to your preferences.
  </Step>
  <Step title="Infraction Types" titleSize="h2">
    The **Infraction Types** fucntions allows you to define different types of infractions, such as Warnings, Terminations, and more. It also includes an Embed Builder that can be used to enhance infraction messages and configure automatic actions.
  </Step>
</Steps>
