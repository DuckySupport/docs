This page provides the essential configuration options for Ducky's **Staff Management** module, which provides an easy way to track disciplinary actions taken on your staff members via infractions, as well as issue promotions. To enable and configure this module, use the **`/setup`** command and navigate to the **Staff Management** page.

<Steps>
  <Step title="Infractions Channel" titleSize="h2">
    The **Infractions Channel** is where all issued infractions are sent. A member receives an infraction when an **ERLC Manager** issues one via the **`/infraction issue`** command. The embeds can be customized to your preference.
  </Step>
  <Step title="Promotions Channel" titleSize="h2">
    The **Promotions Channel** where all promotions are sent. A member receives a promotion when an **ERLC Manager** issues one via the `/promotion issue` command. The embeds can be customized to your preference.
  </Step>
  <Step title="Infraction Types" titleSize="h2">
    The **Infraction Types** are the different types of infractions that can be issued within your server. You can link automated actions, an assigned role, and more via the **Infraction Type Builder**.
  </Step>
</Steps>
