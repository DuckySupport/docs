Duckyâ€™s Discord Moderation feature provides a centralized system to monitor, record, and manage all Discord-related punishments issued through Ducky commands. To enable and configure this feature, use the `/setup` command and select **Discord Moderation**.

<Steps>
  <Step title="Modlogs Channel" titleSize="h2">
    - This is where all moderation actions and log message will be sent.
    - You can configure the **Modlogs Channel** on the **Discord Moderation** page in `/setup`.

    <Tip>
      Choose a private staff channel so only moderators can see these logs.
    </Tip>
  </Step>
  <Step title="Appeal Links (Optional)" titleSize="h2">
    - The **Appeal Link** is an optional URL included in the DM sent to a user when they are moderated.
    - This should direct the user to your chose **appeal** system, such as:
      - **A Discord server invite**
      - **A Google Form**
      - a third-party service like **appeal.gg**

    <Warning>
      If you don't set an **Appeal Link**, no link will be included in the moderation DMs.
    </Warning>
  </Step>
</Steps>
