This page provides the essential configuration options for Ducky's **Feedback** module, which allows your members to submit ratings, reviews, and suggestions for your server. The module creates a centralized system for collecting and managing feedback, ensuring all input is organized and easily accessible. To enable and configure this module, use the **`/setup`** command and navigate to the **Feedback** page.

## Suggestions
    A user can submit a **Suggestion** by using `/suggest`. Members can then Upvote or Downvote your suggestion with the Check and Cross Emojis. Any user who has the **Discord Permission** `Manage Server` will overwrite the votes and Accept or Deny the Suggestion.
## Feedback
    A user can leave a feedback message by using `/feedback submit`. A user must select the  User who they are leaving a feedback on, a rating of 1 to 5 stars. A reason for the feedback is optional.

    The Feedback embed can also be customized to your likings. The Variables can be found in the Embed Builder.
