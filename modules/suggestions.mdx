This page provides the essential configuration options for Ducky's **Suggestion** module, which provides an easy way to track suggestions from the community. To enable and configure this module, use the `/setup` command and navigate to the **Suggestions** page.

<Steps>
  <Step title="Suggestions Channel" icon="hashtag" iconType="regular" titleSize="h2">
    The **Suggestions Channel** is where all suggestions are sent. The channel receives an embed when a member does the `/suggestion submit` command. 

    <Note>
      These embeds can be customized to your liking.
    </Note>
  </Step>
  <Step title="Max Upvotes" icon="up" iconType="regular" titleSize="h2">
    The max upvotes setting is a good way to limit the amount of upvotes.
  </Step>
  <Step title="Max Downvotes" icon="down" iconType="regular" titleSize="h2">
    The **Max Downvotes** setting is a good way to limit the amount of downvotes.
  </Step>
  <Step title="Suggestion Thread" icon="octagon-plus" iconType="regular" titleSize="h2">
    The **Suggestion Thread** is an option you can enable. If checked, Ducky will automattically add a thread. This can be used so members can chat in the thread instead of in the channel.
  </Step>
</Steps>
