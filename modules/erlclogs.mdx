---
description: "Ducky's ERLC Server Logs module allows you to easily keep track of in-game activity, with a simple setup and fully customizable messages."
title: "ERLC Server Logs"
---

<Steps>
  <Step title="Player Join Logs:" titleSize="h2">
    - The \*\*Player Join Logs \*\*sends a log when a user Joins the server, it includes the Roblox Username and a Timestamp of when the user Joined the server.
  </Step>
  <Step title=" Player Leave Logs:" titleSize="h2">
    - The \*\*Player Leave Logs \*\*sends a log when a user leaves the server, it includes the Roblox Username and a Timestamp of when the user Joined the server.
  </Step>
  <Step title="Player Team Logs:" titleSize="h2">
    - The \*\*Player Team Logs \*\*sends a log when a user joins a team. This includes the Team, Callsign, Roblox user and a timestamp.
  </Step>
  <Step title="Player Kill Logs:" titleSize="h2">
    - The \*\*Player Kill Logs \*\*sends a log of when someone gets killed, this includes the user who killed the member and the user who got killed.
  </Step>
  <Step title="Modcall Logs:" titleSize="h2">
    - The \*\*Mod Call Logs \*\*sends a log when a user runs "\!mod", this includes the user who ran the command, it will also send a log when a Mod teleports to a user.
  </Step>
</Steps>
