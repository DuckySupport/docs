Ducky's `Discord Server Status` module allows you to setup stats for `Guild Names`, `Guild IDs`, `Guild Members`, `Guild Bots`, `Guild Humans`(Server Users), `Guild Boosts`, `Guild Boost Level`, `Guild Roles`, `Guild Text Channels`, `Guild Voice Channels`, and `Guild Categories`. Here you you can learn how to create theses stats and how to use the Variables.

### Creating a Status Channel

<Steps>
  <Step title="Getting Your Variable">
    To see all available variables, use the **Variables** button to view the list of variables.

    <Note>
      At least 1 variable is required to create a new channel.
    </Note>
  </Step>
  <Step title="Creating Your Channel">
    To create the actual channel, select the **Create Status Channel** dropdown button. Then, enter the desired channel name with all variables needed.

    <Tip>
      Remember to place curly braces (`{}`) around your variable names.
    </Tip>
  </Step>
</Steps>

## Editing a Status Channel

<Steps>
  <Step title="Edit Status Channel">
    Select **Edit Status Channel** and select the channel you want to edit. Then, enter the desired channel name with all variables needed.

    <Note>
      At least 1 variable is required to edit a new channel.
    </Note>
  </Step>
</Steps>

## Removing a Status Channel

If you'd like to delete a status channel select **Remove Status Channel** and then the channel you'd like to delete.
