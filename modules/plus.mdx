## Plus Profile

Ducky **plus profile** allows you to customize Ducky's name, profile picture, profile banner, and description. To customize Ducky, use the command `/plus profile`. Additionally, there are many presets available for you to chose from, including past Ducky events such as Halloween or Christmas.

## Plus Manage

Ducky's slash command `/plus manage` allows you to purchase Ducky Plus+ slots. This is also where you can apply promo codes which are occasionally given out during the holidays or major Ducky milestones. Additionally, you can use this menu to gift a Ducky Plus+ slot to someone else.

## Plus

Ducky's slash command **`/plus`** allows you to add or remove a Ducky plus slot from your server.

<Note>
  Only the server owner can apply Ducky Plus+ to their server. If you are trying to add Ducky Plus+ to a server you do not own, please gift the server owner a Ducky Plus+ slot.
</Note>
