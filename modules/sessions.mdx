title: "Sessions"
---

This page provides the essential configuration options for **Ducky's Sessions** module, which allows you to easily manage and organize server sessions. The module creates a centralized system for tracking session activity, ensuring all sessions are properly recorded and accessible. To enable and configure this module, use the `/setup` command and navigate to the **Sessions** page.

<Steps>
  <Step title="Sessions Channel" titleSize="h2">
    The **Sessions channel** is where all session messages are sent, they can be send using the commands below.

    - `/session start`
    - `/session end`
    - `/session vote`
    - `/session low`
    - `/session full`

    All of the session embeds are customizable to better suit your server.
  </Step>
  <Step title="Session Mentionables" titleSize="h2">
    The **Session Mentionables** function allows you to select up to **5 roles** that will be mentioned when the session embeds are sent.
  </Step>
  <Step title="Ping Here" titleSize="h2">
    Enabling this function will ping `@here` and the roles you've selected with the function above, if any.
  </Step>
  <Step title="Ping Everyone" titleSize="h2">
    Enabling this function will ping `@everyone` and the roles you've selected, if any.
  </Step>
  <Step title="Server Info" titleSize="h2">
    The **Server Info** includes the Server Name, Max Players and the servers Join Code.
  </Step>
</Steps>
