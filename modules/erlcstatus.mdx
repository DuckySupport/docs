This page provides the essential configuration options for Ducky's **ERLC Server Status** module, which allows you to display live server statistics directly in your Discord server. The module creates a centralized system for monitoring server activity, ensuring key metrics are always visible and up to date. To enable and configure this module, use the **`/setup`** command and navigate to the **ERLC Server Status** page.

<Steps>
  <Step title="Server Status Message" titleSize="h2">
    <Steps>
      <Step title="Channel" icon="hashtag" titleSize="h3">
        The **Channel** is a configurable setting in the **`/setup`** page, this is where your **ERLC Server Status Message** will be sent.
      </Step>
      <Step title="Link" icon="link" titleSize="h3">
        The **Link** option is a non-configurable option which just displays a quick-link to your **ERLC Server Status Message**.
      </Step>
      <Step title="Content" icon="message-dots" titleSize="h3">
        The **Content** setting is where you configure how you want your **ERLC Server Status Message** to look like. You are able to configure the embed and message to your liking.
      </Step>
    </Steps>
  </Step>
  <Step title="Server Status Channels" titleSize="h2">
    With Ducky's **ERLC Server Status** module you are able to configure a variety of status channels to display various types of statistics.\
    To create a status channel, naviage to the **ERLC Server Status** page using the `/setup` command and use the dropdown menu and select "Create Server Status Channel". You are able to create up to **3** Server Status Channels, but you can create up to **6** if you have **Ducky Plus+**.
  </Step>
  <Step title="Variables" titleSize="h2">
    `{server.name}`: The in-game server's name.\
    `{server.players}`: The in-game server's current playercount.\
    **`{server.maxplayers}`**: The in-game server's maximum playercount.\
    `{server.staff}`: The in-game server's current staff count.\
    **`{server.queue}`**: The in-game server's current queue count.\
    **`{server.code}`**: The in-game server's join code.\
    **`{owner.name}`**: The in-game server owner's username.\
    **`{owner.display}`**: The in-game server owner's display name.\
    **`{owner.id}`**:The in-game server owner's Roblox user ID. \
    **`{owner.profile}`**: The link to the in-game server owner's profile.\
    **`{owner.hyperlink}`**:The in-game server owner's username hyperlinked to their profile. \
    **`{team.civilian}`**: The current amount of players on the Civilian team.\
    **`{team.sheriff}`**: The current amount of players on the Sheriff team.\
    **`{team.fire}`**:The current amount of players on the Fire Rescue team. \
    **`{team.police}`**: The current amount of players on the Police team.\
    **`{team.dot}`**: The current amount of players on the DOT team.\
    **`{timestamp}`**: The timestamp of which the server status message was last updated.
  </Step>
</Steps>
