---
title: "Activity Management"
description: "Ducky's Activity Management module allows your staff to request LOAs, and allows for you to ensure your staff's activity."
---

<Steps>
  <Step title="LOA's Enabled" titleSize="h2">
    - Ticking this option to on will enable the LOA function. LOA's can be made when a user runs `/loa request`.
  </Step>
  <Step title="LOA Logs Channel" titleSize="h2">
    - The **LOA Logs Channel** is where a user LOA request gets sent to. A user with the **ERLC Manager Role **accept or deny the request. 
  </Step>
  <Step title="LOA Role" titleSize="h2">
    - The **LOA Role **is assigned to a member once their LOA Request has been approved. 
  </Step>
  <Step title="LOA Nickname" titleSize="h2">
    - The **LOA Nickname **will change the users Nickname after their LOA has been approved. The variables for the LOA Nickname are: 

      `{member.name}`: The name of the member on LOA. 

      `{member.username}`: The username of the member on LOA.

      These Variables can be found on the Dropdown embed.
  </Step>
  <Step title="Activity Report Interval" titleSize="h2">
    - A Duration can be set for the  **Activity Report Interval, **once you've selected a Duration the Activity Report will be generated. You can get the Activity Report by running `/activity report` . You can also use `/activity generate` to quickly generate a Activity Report.
  </Step>
</Steps>
